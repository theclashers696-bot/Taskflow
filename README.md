🚀 Taskflow

A modern, full-stack task and team management platform built with Next.js 15, TypeScript, Prisma, PostgreSQL (Neon), and Better Auth.

«Organize tasks, collaborate with teams, and boost productivity with a clean, responsive interface.»

🌐 Live Demo

https://taskflow-six-lovat.vercel.app



✨ Features

- 🔐 Secure Authentication (Better Auth)
- 👤 User Registration & Login
- 📋 Task Management
- 📌 Kanban Board
- 👥 Team Management
- 📊 Dashboard Analytics
- 🔔 Notifications
- ⚙️ User Settings
- 👤 Profile Management
- 📱 Fully Responsive Design
- 🌙 Modern UI with Tailwind CSS

🚀 Getting Started

Clone the repository

git clone https://github.com/theclashers696-bot/Taskflow.git

cd Taskflow

Install dependencies

pnpm install

or

npm install


Environment Variables

Create a ".env" file.

DATABASE_URL=

DIRECT_URL=

BETTER_AUTH_SECRET=

BETTER_AUTH_URL=http://localhost:3000

NEXT_PUBLIC_APP_URL=http://localhost:3000



Push Prisma Schema
npx prisma db push


Run Development Server

pnpm dev
or
npm run dev

Open:
http://localhost:3000

Build
pnpm build
